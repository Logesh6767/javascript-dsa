### 4. **API Routes in Next.js** (Week 4, Day 25)

- Request/Response handling
- Middleware patterns
- Error handling

- Module system (CommonJS vs ES Modules)
- Built-in modules (fs, path, http)
- Async operations in Node
- Creating simple servers
